SCHUHHAUS WOLF — Luxury Footwear Atelier
A Digital Remake inspired by Magnetism.fr
This project is a high-end redesign of the traditional Schuhhaus Wolf website. It transitions the brand from a local retail aesthetic to a luxury-focused, minimalist digital experience specializing in premium European craftsmanship.

✦ Design Philosophy
Inspired by the Magnetism creative agency, this site utilizes:

Brutalist Typography: High-contrast, oversized serif fonts for a bold brand presence.

Immersive Motion: Interactive 3D elements and smooth parallax scrolling.

Minimalist Layout: Deep focus on negative space to highlight premium brands like AGL, Premiata, and Voile Blanche.

✦ Key Features
Brand Tribute Intro: A high-speed scrolling animation showcasing the store's luxury portfolio.

Interactive Hero: A 3D floating footwear element that reacts to the user's cursor movement.

Dynamic Grid: An editorial-style project grid for featured collections.

Follower Cursor: A custom-designed cursor for a fluid, high-end browsing experience.

✦ Tech Stack
Frontend: HTML5 / CSS3 / Vanilla JavaScript

Typography: Google Fonts (Inter)

Animations: CSS Keyframes & Intersection Observer API
